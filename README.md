# iwama
Photogrammetric model taken with an iPhone.  
iPhoneでフォトグラメトリした [iwama](https://twitter.com/iwamah1) の3Dモデル. 

iwama.fbx : No weights and bones. 
iwama_weight.fbx : Weights and bones available.
